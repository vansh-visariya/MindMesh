# Technology Stack & News Sources

## Technology Stack
- **Backend Framework**: FastAPI (Python)
- **Web Scraping**: Scrapy, Requests, BeautifulSoup4
- **Database**: PostgreSQL (Production), SQLite (Dev/Testing if needed)
- **ORM**: SQLAlchemy
- **Frontend**: HTML, CSS, JavaScript (Vanilla/Bootstrap)
- **AI/LLM**: Groq API (or similar LLM provider)
- **Task Scheduling**: APScheduler (or system cron)
